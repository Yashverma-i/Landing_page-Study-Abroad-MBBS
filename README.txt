This landing page aims to provide prospective medical students with detailed information about studying MBBS abroad. The page includes various sections such as:

Hero Section: Engaging banner with a Call to Action (CTA) to apply for MBBS abroad.

Why Study MBBS Abroad: Key benefits of studying MBBS abroad.

Top Countries: Country listings including Russia, Uzbekistan, Kazakhstan, Philippines, Georgia, Kyrgyzstan, and Egypt.

Admission Process & Eligibility: Steps to apply and eligibility criteria.

Call-to-Action (CTA): A lead generation form where users can input their name, email, phone, and country preference to inquire more about programs.

Features-------------------------
1. Fully responsive design for optimal viewing on desktop, tablet, and mobile devices.
2. Clean, modern UI with detailed sections about each country.
3. Lead generation form to capture user details.
4. Interactive and user-friendly interface with smooth animations and transitions.
5. SEO optimized and fast loading for better user experience


Technologies Used-----------------------------------------
HTML: For structuring the webpage.
CSS: For styling and creating the responsive layout.
JavaScript: For handling form submissions and animations.
Bootstrap : For responsive design and grid layout.
Font Awesome: For icons.
